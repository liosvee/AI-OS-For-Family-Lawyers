# Matter Templates

Matter Templates accelerate matter setup and improve consistency.

## Example Templates

- Divorce
- Custody
- Child Support
- Spousal Support
- Property Division
- Enforcement
- Modification
- Relocation
- Paternity
- Protective Orders

## Template Components

Templates may pre-build:

- intake questions
- issue categories
- document request categories
- witness buckets
- timeline types
- evidence categories
- draft output types
