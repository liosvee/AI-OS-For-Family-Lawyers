# Code of Conduct

## Our Standard

This project is intended to support professional legal workflow systems. We expect contributors and participants to engage respectfully, constructively, and professionally.

Examples of expected behavior include:

- using respectful and professional language
- providing constructive feedback
- focusing on problem-solving and collaboration
- respecting confidentiality and security concerns
- being transparent about limitations and uncertainty

Examples of unacceptable behavior include:

- harassment or abusive conduct
- discriminatory language or behavior
- bad-faith disruption
- intentional misrepresentation
- encouraging unethical or unlawful conduct
- posting sensitive information irresponsibly

---

## Project Context

Because this repository concerns legal-workflow infrastructure, contributors should take particular care to avoid:

- exaggerated claims about legal capability
- unsafe advice regarding confidential information
- advocacy for deceptive or unlawful behavior
- dismissive treatment of security or ethical review requirements

---

## Enforcement

Project maintainers may remove, edit, or reject contributions or participation that conflict with this Code of Conduct or undermine the project’s legal, ethical, or security standards.
