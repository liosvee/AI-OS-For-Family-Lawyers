# Client Document Request Portal

The Client Document Request Portal produces structured, dynamic document request lists tailored to each matter.

## Common Requests

- pay stubs
- tax returns
- bank statements
- retirement statements
- credit card statements
- property records
- mortgage statements
- business financial records
- parenting logs
- school records
- medical records
- prior court orders

## Workflow Goals

- track requested vs. received records
- identify missing categories
- align collection with discovery planning
- update requests as new issues arise
