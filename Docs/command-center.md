# Case Command Center

The Case Command Center is the central workspace for each matter.

## Recommended Sections

- Matter Summary
- Triage Snapshot
- Key Facts
- Timeline
- Legal Issues
- Evidence Matrix
- Witness Tracker
- Financial Snapshot
- Discovery Plan
- Deadlines
- Drafting Queue
- Risk Flags

## Purpose

The Command Center should give the legal team a single operational view of what is known, what is disputed, what must be proven, and what must happen next.
